Customer Review Classification

  • Category: Natural Language Processing
  • Client: Axon
  • Project date: Jan, 2023 - May, 2023

Summary

  • The project aims to improve Axon's NPS score by developing a machine learning model to categorize customer feedback and generate reports for areas of improvement. The model uses word embeddings and additional features, utilizing Logistic Regression with L2 Regularization, and resulted in better customer satisfaction and NPS score.

  • Background: Axon is a leading provider of public safety technology solutions, focusing on customer experience and using NPS as a key metric. The goal was to automate the feedback analysis process to enhance customer satisfaction and improve the NPS score.

  • Models Used: Logistic Regression with L2 Regularization, SMOTE, Word2Vec (word embeddings).

  • Methodology: The project involved data exploration, cleaning, pre-processing, and feature selection. Word embeddings were used for text vectorization, and SMOTE was employed for handling imbalanced data. Logistic Regression with L2 Regularization was chosen for classification, and hyperparameter tuning was performed using randomized search.

  • Conclusion: The developed machine learning model with word embeddings and additional features significantly improved NPS scores by accurately categorizing customer feedback. Logistic Regression with L2 Regularization emerged as the best-performing model for the task, aiding Axon in making better business decisions and enhancing customer experience. .
